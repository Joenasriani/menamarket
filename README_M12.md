# MENAMarket M12 overlay — Public API Expansion

Apply this overlay on top of M11.

Purpose:
- add a broader read-only public market API
- expose list, detail, resolution, lifecycle, and discovery endpoints
- support filtering, pagination, and clean JSON output
- keep all data file-backed and truthful

This stage still does not add:
- user auth
- trading execution
- payments
- websocket streaming
