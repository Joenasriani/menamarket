# MENAMarket M07 overlay — Public Discovery

Apply this overlay on top of M06.

Purpose:
- add real public discovery features for published markets
- support search, category filters, jurisdiction filters, and status filters
- add a categories API based on the actual market catalog
- keep all discovery views driven by real file-backed market records only

This stage still does **not** add:
- fake trending modules
- fake prices
- fake order books
- wallet flows
- settlement
- funding
- recommendation engines
