# 07_DESIGN_SYSTEM

## Design intent
MENAMarket should feel clean, contemporary, and original.
It must not visually imitate Polymarket branding.

## Initial principles
- dark-first interface is allowed but not required
- typography must be highly readable
- cards, tables, and badges should be consistent
- empty states and error states must be designed, not ignored
- loading states must be explicit
- market data presentation must favor clarity over visual noise

## Base tokens
- semantic colors only
- spacing scale
- radius scale
- shadow scale
- type scale
- z-index system
